🚀 AI-Based Sales Representative CRM
An intelligent, full-stack CRM system that uses AI to track user behavior, score leads, and automate sales follow-ups — helping businesses convert more customers with data-driven decisions.

📌 Overview
This project is a SaaS-based CRM platform built with:
⚛️ React (Frontend)
🐍 Django + Django REST Framework (Backend)
🗄️ MySQL (Database)
It simulates a real-world sales ecosystem where:
Each company has its own isolated dashboard
A tracking SDK captures user activity from client websites
AI analyzes behavior to generate lead scores
Sales reps get smart follow-up suggestions

🎯 Key Features
🔐 Authentication System
JWT-based login/signup
Secure session handling
Role-based access (Admin / Sales Rep)
🏢 Company-Based Multi-Tenancy
Each company gets:
Isolated data
Separate dashboards
Independent users

📊 Dashboard Analytics
Lead conversion stats
User engagement metrics
Sales performance tracking
Visual charts & insights

👥 Lead Management System
View all leads in tabular format
Detailed lead profile pages
Track:
User activity
Interaction history
Conversion probability

🌐 Website Tracking SDK
JavaScript SDK integrated into client websites
Tracks:
Page visits
Click behavior
Time spent
Sends data to backend in real-time

🤖 AI Lead Scoring Engine
Assigns scores based on:
User activity
Engagement level
Behavioral patterns
Helps prioritize high-value leads

📞 Smart Follow-Up System
AI suggests:
When to contact a lead
What action to take
Improves conversion efficiency

⚙️ Profile & Settings
Company-specific configuration
User profile management
